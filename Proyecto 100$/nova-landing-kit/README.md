## ✨ Features

- **25+ Premium Components** — Buttons, cards, forms, navigation, pricing tables, testimonials, FAQ, stats, alerts, badges, loaders, and more
- **Zero Dependencies** — Pure HTML, CSS, JavaScript. No build tools, no npm, no frameworks required
- **Fully Responsive** — Every section adapts perfectly from mobile to 4K
- **Dark Theme** — Modern dark design with carefully crafted gradients and typography
- **Smooth Animations** — Intersection Observer reveals, animated counters, floating elements
- **CSS Variables** — Easy to customize colors, spacing, fonts, and more
- **Copy-Paste Ready** — Each component is self-contained with its own styles

## 🚀 Quick Start

1. Open `index.html` in your browser
2. Copy the sections you want
3. Paste into your project
4. Customize the CSS variables to match your brand

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800;900&display=swap" rel="stylesheet">
```

## 🎨 Customization

All design tokens are controlled via CSS custom properties in `:root`. Change the entire look by modifying just a few variables:

```css
:root {
  --accent: #6c5ce7;        /* Primary accent color */
  --grad-1: #6c5ce7;        /* Gradient start */
  --grad-3: #fd79a8;        /* Gradient end */
  --bg: #07070d;             /* Background color */
  --text: #eeeef2;           /* Text color */
}
```
