❓ FAQ

Frequently asked questions

What do I get when I purchase? ▼

You get the complete Nova UI landing page kit as a fully functional HTML file, including all CSS, JavaScript, and component documentation. Everything is self-contained — no external dependencies required.

Do I need to know React, Vue, or any framework? ▼

No! Nova UI is built with vanilla HTML, CSS, and JavaScript. It works with any framework or no framework at all. If you know basic HTML, you can use this kit.

Can I use this for commercial projects? ▼

Yes! The Professional tier includes a commercial license. You can use the components in unlimited projects, including client work. The Enterprise tier adds white-label rights.

What if I'm not satisfied? ▼

We offer a 14-day money-back guarantee. If the kit doesn't meet your expectations, contact us and we'll refund your purchase — no questions asked.

Will I get updates? ▼

Yes! Professional and Enterprise tiers include lifetime updates. We regularly add new components and improvements based on user feedback.
